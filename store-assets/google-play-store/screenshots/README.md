# Google Play Store Screenshots

## 📱 Where to Put Your Screenshots

Add **1–8 screenshots** of your app here. Each must be:
- **Dimensions:** 1080 × 1920 pixels
- **Format:** PNG or JPG
- **Portrait orientation**

## 🎯 Recommended Screenshot Sequence

1. **main-screen.png** (1080×1920)
   - Shows the Qur'an recitation screen
   - Displays word-by-word highlighting
   - Caption: "Melodic Recitation 🎙️"

2. **voice-search.png** (1080×1920)
   - Shows Voice Assistant or Surah Selection
   - Caption: "Voice Search 🔍"

3. **reciter-select.png** (1080×1920)
   - Shows reciter picker (6 reciters available)
   - Caption: "6 Renowned Reciters 🎵"

4. **translation.png** (1080×1920)
   - Shows English translation or "Both" mode
   - Caption: "English Narration 🗣️"

5. **bookmarks.png** (1080×1920)
   - Shows bookmarked verses
   - Caption: "Save & Resume 📖"

6. **settings.png** (1080×1920)
   - Shows voice settings, profiles, speed control
   - Caption: "Customize Your Experience ⚙️"

7. **voice-practice.png** (1080×1920)
   - Shows voice recording/practice mode
   - Caption: "Practice Your Recitation 🎤"

8. **themes.png** (1080×1920)
   - Shows day/night modes, themes
   - Caption: "Day/Night Mode 🌙"

## 🎨 How to Create Screenshots

### Option 1: Android Emulator (Easiest)
```bash
# Start the app in Android emulator at 1080×1920 resolution
npm start
# Select Android emulator

# In Android emulator: Menu → Screenshot
# Or press Ctrl+Shift+S (Android Studio)
# Save as: 01-main-screen.png
```

### Option 2: Physical Device
- Use developer tools or built-in screenshot (Power + Volume Down)
- Connect via USB to computer
- Copy screenshots from device storage

### Option 3: Add Text Overlays (Recommended)
1. Take screenshots (using options above)
2. Open in Figma, Photoshop, or free tools (GIMP, Pixlr)
3. Add text labels in top-left or bottom-center (readable size)
4. Export at 1080×1920 PNG

## 🚀 Upload to Google Play Store

1. Go to [Google Play Console](https://play.google.com/console)
2. Select your app → **Store presence** → **Main store listing**
3. Scroll to **Screenshots** section
4. Upload images (1–8)
5. Drag to reorder (first screenshot is most important!)
6. Save and publish

## 💡 Pro Tips

- **Screenshot #1 is critical** — it's the largest preview users see
- **Keep text consistent** — use the same font/style across all
- **Feature your best screens** — lead with word-by-word, voice search, or reciter selection
- **A/B test if you can** — some stores allow testing different screenshot sets
- **Update regularly** — if UI changes, update screenshots
